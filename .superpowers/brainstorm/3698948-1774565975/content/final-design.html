Final Design: Subtitle Preview/Editor

Revised after 3 rounds of review across 10 specialized agents

Architecture

Module 1

SubtitleDocument

Pure data layer. 8 format parsers (tiered testing). Immutable cue ops + undo/redo command stack.

Module 2

SubtitleEditor

Virtualized cue table (role="grid"), toolbar (role="toolbar"), reference column, upload/download.

Module 3

VideoPreview

Vidstack player + server-side FFmpeg remux. JASSUB for ASS/SSA. No client WASM remux.

Key Design Changes from Review

REMOVED
  • Client-side WASM remux (libav.js)
  • Raw filesystem paths in API
  • 60fps state in React context
  • Unvirtualized cue table
ADDED
  • Server-side FFmpeg remux to fMP4
  • ETag optimistic locking for saves
  • Collapsible/resizable video panel
  • Live timing mode (I/O keys)
  • Keyboard-first editing (F2, Shift+F10)
  • ARIA live regions, role="grid"

Phased Delivery

PHASE 1

Core Editor

~3500-4000 lines

  • All 8 format parsers
  • Virtualized cue table
  • Inline text + timing editing
  • Add/delete/split/merge cues
  • Undo/redo
  • Reference subtitle column
  • Upload/download/create new
  • Save with ETag locking
PHASE 2

Video Playback

~2500-3000 lines

  • FFmpeg remux endpoint
  • Vidstack player (collapsible)
  • JASSUB for ASS/SSA overlay
  • SRT-to-VTT native track
  • Playhead sync (ref + rAF)
  • Click-cue-to-seek
  • Auto-scroll (off by default)
PHASE 3

Timeline + Live Timing

~2000-2500 lines

  • Canvas-based timeline strip
  • Drag-to-resize/move cues
  • Zoom (buttons + scroll)
  • Shift-all-timings tool
  • Live timing mode (I/O keys)

Backend API (ID-based, no raw paths)

GET /api/episodes/{id}/subtitles/{sub_id}/content
PUT /api/episodes/{id}/subtitles/{sub_id}/content + If-Match
POST /api/episodes/{id}/subtitles (create new)
GET /api/episodes/{id}/preview?start=600&duration=120 (FFmpeg remux)
GET /api/movies/{id}/subtitles/{sub_id}/content
PUT /api/movies/{id}/subtitles/{sub_id}/content + If-Match
POST /api/movies/{id}/subtitles
GET /api/movies/{id}/preview?start=600&duration=120
